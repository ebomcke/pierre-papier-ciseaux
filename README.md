# Pierre Papier Ciseaux (Pygame)

A simple game project using Python and Pygame.

## Setup

1. Install [Poetry](https://python-poetry.org/docs/#installation).
2. Install dependencies:
   ```bash
   poetry install
   ```

## Run the game

```bash
poetry run ppc-game
```
